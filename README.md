# Design-TearDown

This project is about cloning/building a heatmap of the Smashing magazine website using HTML5/CSS3

![screenshot](./img/project-screenshot.png)

## Built With

HTML

- Designed in Semantic HTML style.

CSS

- Use of Gradient backgound.
- Use of flex layout
- Use of grid layout
- Use of float
